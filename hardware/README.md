# Hardware

This directory contains the hardware documentation for the Vehicle Gateway ECU.

## Hardware Modules

- NXP i.MX 8M Plus EVK
- STM32 NUCLEO-G071RB
- Potentiometers
- DS18B20 Temperature Sensor
- USB Communication
- UART Interface

## Documents

| File | Description |
|------|-------------|
| COMPONENTS.md | Hardware components |
| PIN_CONNECTIONS.md | Pin mapping |
| POWER_REQUIREMENTS.md | Power requirements |

The hardware is designed to demonstrate a Vehicle Gateway ECU capable of acquiring vehicle parameters and displaying them on an Embedded Linux dashboard.
