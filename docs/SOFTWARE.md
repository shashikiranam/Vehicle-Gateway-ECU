# Software

## Operating System

- Embedded Linux

---

## Firmware

### STM32

Responsible for:

- Reading ADC values
- Acquiring vehicle parameters
- Sensor interfacing
- UART transmission

---

### NXP i.MX 8M Plus

Responsible for:

- UART reception
- Data parsing
- Dashboard execution
- Vehicle diagnostics
- Data logging
