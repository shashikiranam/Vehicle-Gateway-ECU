# Documentation

This directory contains the complete technical documentation for the Vehicle Gateway ECU project.

| File | Description |

| SYSTEM_ARCHITECTURE.md | Overall system architecture |

The project demonstrates an Automotive Vehicle Gateway ECU using the NXP i.MX 8M Plus EVK and STM32 microcontroller for real-time vehicle data acquisition, communication, diagnostics, and visualization.
