# Project Overview

## Project Title

Vehicle Gateway ECU

## Objective

Develop an Automotive Vehicle Gateway ECU capable of collecting vehicle parameters from an STM32 microcontroller and displaying them on an Embedded Linux dashboard running on the NXP i.MX 8M Plus EVK.

## Vehicle Parameters

- Vehicle Speed
- Engine RPM
- Fuel Level
- Engine Temperature

## Functionalities

- Real-time data acquisition
- UART communication
- CAN-ready architecture
- Embedded Linux dashboard
- Data logging
- Diagnostics
- Fault monitoring

## Hardware

- NXP i.MX 8M Plus EVK
- STM32 NUCLEO-G071RB
- Potentiometers
- DS18B20 Temperature Sensor

## Software

- Embedded C
- C++
- Python
- Embedded Linux
