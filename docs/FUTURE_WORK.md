## Long-Term Vision

The current Vehicle Gateway ECU serves as a learning and portfolio project. Future versions may evolve into a complete automotive gateway capable of integrating multiple Electronic Control Units (ECUs), communication protocols, and cloud connectivity while supporting advanced diagnostics and vehicle monitoring.
