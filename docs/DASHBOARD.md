# Dashboard

## Overview

The Embedded Linux dashboard running on the NXP i.MX 8M Plus EVK provides real-time visualization of vehicle parameters received from the STM32 microcontroller.

---

## Displayed Parameters

- Vehicle Speed (km/h)
- Engine RPM
- Fuel Level (%)
- Engine Temperature (°C)
- UART Communication Status

---

## Features

- Real-time updates
- Simple terminal-based dashboard
- Vehicle status monitoring
- Communication status indicator
- Expandable for GUI implementation
